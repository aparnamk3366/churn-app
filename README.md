# Customer Churn Prediction App

This is a machine learning project that predicts whether a customer will churn or not.

## Features
- User-friendly web app built using Streamlit
- Predicts customer churn based on input data
- Uses trained machine learning model (Logistic Regression / other model)

## Files
- app.py → Main Streamlit application
- requirements.txt → Required Python libraries
- model file → Trained ML model (if included)
